# Ahmed Bin Athar - Portfolio

This is the source code for my personal portfolio website, showcasing my work in Filmmaking, AI, and Marketing.

## Features
- **Responsive Design**: Mobile-first layout using CSS Grid and Flexbox.
- **Dark Mode**: Toggle between light and dark themes.
- **YouTube Integration**: Embedded video players for films and channel.
- **Linktree Integration**: Prominent links to social profiles.
- **SEO Optimized**: Meta tags, Open Graph, and semantic HTML.

## Setup & Deployment

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/ahmedrikk/ahmedrikk.github.io.git
   ```
2. Open `index.html` in your browser.

### Deployment to GitHub Pages
This repository is configured with a GitHub Actions workflow to automatically deploy to GitHub Pages.

1. Push your changes to the `main` branch.
2. Go to your repository **Settings** > **Pages**.
3. Ensure **Source** is set to **GitHub Actions**.
4. The workflow will run and deploy your site.

## Customization
- **Resume**: Replace `assets/resume.pdf` with your actual PDF resume.
- **Content**: Edit `index.html` to update text and links.
- **Styles**: Modify `css/style.css` to change colors and layout.
